Installation manual

Requirements

  • Webserver such as Apache, Nginx, IIS, etc
  • PHP 6.0
  • MSQL 5.6
  • A cup of good coffee

Installation

  1. You need create a database on your sql server if you don’t know how this Google it.
  2. Configure “config.php” to reflect your MySQL connection string.
  3. Use the install script to make the database ready. Follow al the steps on the screen from left to right
    1. In Step one you can check if your server meets the requirements.
    2. By clicking the button in step 2 all of the database table that are needed.
    3. At step 3 you can make and insert one or more admin accounts.
  4. Try to login if it doesn’t work try again.
  5. If all was successful please remove the complete “install” folder.
